 Restaurant Analytics Capstone

 Project Overview
Analysis of restaurant sales data to identify peak hours, top-selling items, and revenue trends.

 Tools Used
- Excel (Cleaning & Dashboard)
- SQL (Data extraction & insights)
- Power BI (Interactive visualization)
- GitHub (Version control)

 Dataset Description
Restaurant order transactions including Order ID, Date, Item, Category, Price, and Payment Type.

 Cleaning Process
1. Removed duplicate entries
2. Standardized date formats
3. Filled missing Category values
4. Removed outlier transactions

 Key Insights
- Peak hour: 1:00 PM – 2:00 PM accounts for 25% of daily revenue
- Top item: Chicken Burger contributes 15% of total sales
- 70% of orders use digital wallets
